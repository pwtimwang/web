== Description ==
Install this plugin,You can show a excerpt on your weblog homepage,You can  omit the "more" tags.
My English is poor,Maybe you should read this page : http://www.kutailang.com/?page_id=397

== Installation ==
一、解压wp-chinese-excerpt.zip，然后将wp-chinese-excerpt文件夹上传至插件目录(也就是你的网站  `/wp-content/plugins/`  目录内)；或者登陆后台在后台菜单中选择添加插件，然后上传wp-chinese-excerpt.zip并安装。
二、登陆后台激活改插件
三、在后才 设置-摘要设置 页面中设置几个参数

 == Translations ==
 此插件暂时1.0版本并没有提供语言文件，因为除了作者只会中文和英文，英文的话又不需要插件，而其他语言作者没法测试
